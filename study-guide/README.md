# C++ Study Guide App

This is a zero-dependency static app built on top of the repository contents.

## Run locally

From the repository root:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/study-guide/
```

## What it includes

- Chapter-by-chapter study modules based on the `Chapter*` folders
- Practice exercises for every study module
- Source preview buttons that load real `.cpp` files from the repo
- Repository overview showing how to use `Programming Challenges`, `CSC111_ProfYan`, and `CSC211H_Prof_Azhar`
- Cleanup recommendations for binaries and temporary files
